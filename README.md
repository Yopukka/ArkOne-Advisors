# ArkOne Advisors 🏢

Commercial real estate landing page built with **HTML, CSS and vanilla JavaScript**, currently live in production.

🔗 **[View Live Site](https://arkoneadvisors.com)**

## ✨ Features

- Crossfade background video hero section (30s interval)
- Responsive navbar — side menu on mobile, top bar on scroll
- Service cards with dynamic background image on hover
- Testimonials carousel (Bootstrap)
- Calendly booking widget integration
- Scroll-triggered animations
- Mobile-first responsive design

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantics |
| CSS3 | Styling & animations |
| JavaScript (Vanilla) | DOM manipulation, scroll events, video crossfade |
| Bootstrap 4 | Grid system & carousel |
| Font Awesome | Icons |
| Google Fonts | Typography (Mulish, Space Grotesk) |
| Calendly Widget | Booking integration |

## 🚀 Getting Started

No dependencies or build tools required.

```bash
git clone https://github.com/Yopukka/arkone-advisors.git
cd arkone-advisors
open index.html
```

Or simply drag `index.html` into your browser.
